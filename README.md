# rate-json (default) — Tomanify JSON Feed

This repository publishes a **free JSON snapshot of Iran free-market currency rates (Toman)** for:
**USD, EUR, AED, TRY, CNY**.

It is maintained by **Amin Raoufi** and primarily powers the **Tomanify** WordPress/WooCommerce plugin:
- Plugin: https://wordpress.org/plugins/tomanify

---

## What is this?
A lightweight, public **JSON data feed** intended for developers and website owners who need a simple, fast endpoint for exchange-rate snapshots.

- **Update frequency:** 3-4× per day
- **Pipeline:** rates are collected on an external server, cross-checked/validated, and a final consolidated snapshot is published here.

---

## Data
- Base unit: **Toman**
- Currencies included: **USD / EUR / AED / TRY / CNY**
- File: `data.json`

> Note: The exact JSON fields may evolve, but currency values are always provided under stable currency codes.
